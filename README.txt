# Student Management App

In this repo we will create different types of Apps in .NET Like:
	1. Console App
	2. MVC Web App
	3. WPF (Windows/Desktop App)
	4. ASP.NET Web APIs (Angular Frontend)

Database Communication Approaches:
	1. Entity Framework Core (EF Core)
	2. Dapper 
	3. ADO.NET
	4. Stored Procedures

We Will use a Single (Shared) Database for all the Apps.
